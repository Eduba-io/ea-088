# Delta Air Lines Inc. Landing Page

Single-page, static landing page prepared for Delta Air Lines, Inc. ahead of a conference booth conversation. The page carries a short read on where AI fits in Delta's stack, anchored on three concrete reads (Fetcherr governance, post-CrowdStrike resiliency, and TechOps documentation), and routes to a Calendly booking with Matt Creamer (CRO, Eduba). Visual system, palette, and typography are drawn from the Delta brand pack (crimson #C8102E, regal blue #003A70, burnt umber #862633, Whitney with Helvetica Neue and Arial fallbacks). Eduba branding appears only in the footer and in the named case study references, per the shared chrome rules. No external webfonts are loaded so the page renders in the same stack Delta uses on its own surfaces.

Intended URL path: /for/delta-air-lines-inc
